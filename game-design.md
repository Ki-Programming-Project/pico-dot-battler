# Pico Dot Battler - Game Design Document
version: 0.1.0 / 2026-03-19

---

## 1. コンセプト

- プレイヤーが16×16のドット絵を描き、そのキャラクターでCPUと戦う育成バトルゲーム
- 色の種類・ピクセル数からステータスを自動算出
- HTML/CSS/JS単一ファイル構成、スマホ（360×800px）基準

## 5. ドット絵エディタ

- グリッド：16×16px
- 操作：指なぞり描き（touchmove）
- 初期カラー：4色（白・黒・赤・青）
- アンドゥ：最大10手順
- 保存上限：初期5体（ショップで拡張可・将来実装）
- キャラ名：色＋体型ワードで自動生成→変更可
- 削除制約：出撃中・1体のみの場合は削除不可
- 複製：ドット絵のみコピー（レベル・装備リセット）
- お気に入り：上限なし（ソート条件追加・削除保護）

## 6. ステータス変換ルール

計算式：`ステータス値 = 色の種類数 × ピクセル数 × 係数 × 基礎倍率`

| 色系統 | ステータス | 係数 |
|---|---|---|
| 赤・オレンジ | ATK | 1.5 |
| 青・水色 | DEF | 1.2 |
| 緑 | HP | 3.0 |
| 紫・ピンク | SPD | 1.0 |
| 黄 | SP回復 | 0.8 |
| 白 | SP回復 | 0.5 |
| 黒 | DEF | 0.8 |
| 灰・その他 | バランス補正 | 0.3 |

基礎倍率：`総塗りピクセル数 / 256 × 1.5`

## 7. スキル自動決定

最高ステータスに応じてスキルセットが決まる。

| 最高ステータス | スキル1（初期） | スキル2（Lv5） | スキル3（Lv10） |
|---|---|---|---|
| ATK | 強攻撃×2.5 | 連続攻撃×1.2×3 | 必殺×4.0 |
| DEF | 防御バフ2T | カウンター | 無敵1T |
| HP | 自己回復30% | HP吸収攻撃 | 完全回復 |
| SPD | 先制攻撃 | 2連行動 | 回避バフ |
| SP回復 | SP全回復 | 全能力バフ | SP消費なし3T |

スキルエフェクト色：
- 攻撃系：赤（敵は暗い赤）
- 回復系：緑（敵は暗い緑）
- バフ系：黄（敵は暗い黄）

## 8. バトルシステム

### 基本
- 完全オート進行（1秒/ターン）
- SPゲージ満タンでスキルボタン発光→タップ発動
- スピード：1倍→2倍→4倍→1倍（タップで循環）
- ライフ制：残機3（ハートアイコン♥♥♥）
- ターン上限：100ターン（HP多い方が勝利）
- 引き分け：両者ライフ−1・スコア半分

### ダメージ計算
```
基本ダメージ = ATK − DEF × 0.5
乱数補正    = ±10%
クリティカル = SPD差に応じて確率上昇（最大50%）・ダメージ×1.5
```

### SPゲージ
- 初期値：0
- 毎ターン SP回復値 ずつ増加
- スキル発動後：0にリセット
- SP不足：グレーアウト
- クールダウン中：残りターン数をボタン上に表示

### 先攻決定
- SPD比較（高い方が先攻）
- 同値の場合はランダム

### 敵の行動
- 一定確率でスキル使用（プレイヤーと同じルール）
- HP30%以下：ATK×1.5（強化状態）

### ステージ
毎戦ランダム抽選（5種）

| ステージ | ステータス補正 | ギミック | ドロップ素材 |
|---|---|---|---|
| 草原 | HP+10% | なし | HP素材 |
| 洞窟 | DEF+10% | 毎5T落石（全体5ダメ） | DEF素材 |
| 城 | ATK+10% | なし | ATK素材 |
| 海岸 | SPD+10% | 毎3T波（SPゲージ+20%） | SPD素材 |
| 火山 | ATK+15% | 毎4T噴火（両者5ダメ） | SP回復素材 |

バトル開始時：ステージ名・補正・ギミック説明を3秒表示（タップでスキップ）

### 演出
- 攻撃時：キャラが0.2秒スライド
- ダメージ：数値テキストが浮かび上がる
- スキル発動：画面フラッシュ＋色付きエフェクト
- 敗北：キャラが下に落下（0.5秒）→残機あり復活（0.3秒）
- 勝利：演出後1.5秒で自動リザルト遷移
- アイドル：上下2px・1秒周期で揺れ
- SE：Web Audio APIビープ音（外部ファイルなし）

### SE一覧
| 種別 | 周波数 | 長さ |
|---|---|---|
| attack | 220Hz | 0.1秒 |
| skill | 220→440Hz | 0.3秒 |
| damage | 110Hz | 0.1秒 |
| victory | 上昇3音 | 0.5秒 |
| defeat | 下降3音 | 0.8秒 |
| levelup | 明るい3音 | 0.6秒 |

BGM：将来実装のためフック関数のみ用意

## 9. スコア・ランキング

```
スコア = 撃破数 × 残HP率(%) × ターンボーナス × 連勝倍率
ターンボーナス = 20T以内:×2.0 / 40T以内:×1.5 / 以降:×1.0
連勝倍率 = 1.0〜2.0（連勝数に応じて上昇）
連勝リセット = 敗北 or 離脱
```

ランキング表示項目：
- 順位 / スコア / キャラ名 / ドット絵（16×16縮小）
- 最大連勝数 / 総撃破数 / 最長生存ターン数

## 10. 育成システム

### レベル
- 上限：Lv50
- 獲得EXP = 敵の(ATK+DEF+HP+SPD) ÷ 4 × 難易度係数
- 難易度係数：1〜5戦:1.0 / 6〜15戦:1.5 / 16戦〜:2.0
- Lv50到達後：経験値→ポイント変換

### スキル解放
- Lv5：スキル2解放
- Lv10：スキル3解放

### 進化
- 条件：Lv20到達＋専用アイテム消費（ショップ500pt）
- 進化後：Lv20のまま継続（上限50まで）
- ドット絵：既存に持っている色のみでランダム数ピクセル追加→編集開始

### 装備
- スロット：武器・防具・アクセの3枠
- 所持上限：各スロット10個まで
- レア度別強さ：
  - ノーマル：フラット値（ATK+8など）/ ドロップ率60%
  - レア：パーセント（ATK+8%など）/ ドロップ率30%
  - 超レア：両方混在（ATK+10かつ+5%）/ ドロップ率10%
- 装備強化：+1〜+10（コスト：素材＋ポイント、+1:50pt→+10:500pt）
- 装備比較：タップで現在装備との差分表示（+/-）

### 強化素材
- ドロップ：バトル後一定確率（装備と同抽選）
- ステージ別：草原→HP素材・洞窟→DEF素材・城→ATK素材・海岸→SPD素材・火山→SP回復素材
- 所持上限：各素材99個
